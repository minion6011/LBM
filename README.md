# LBM - Lobby Bot Maker 
**LBM** is a Lobby Bot network that was created in `2021` and remained operational and updated until `2026`.
> A Lobby bot is a bot that allows you to "view skins you don't own" on your Fortnite account by creating a second account as a bot and having it enter the Lobby using the appropriate commands.
